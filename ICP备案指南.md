# ICP 备案操作指南

> 目标：为 miaotools.com.cn 完成 ICP 备案，解决百度收录问题

---

## 前置条件：迁移到腾讯云 CloudBase

**重要**：ICP 备案必须填写国内有 ISP 许可证的 hosting 提供商。GitHub Pages 是海外服务器，无法用于备案。

### 为什么需要迁移

| 对比项 | GitHub Pages | 腾讯云 CloudBase |
|--------|-------------|-----------------|
| 服务器位置 | 美国 | 中国国内 |
| ISP 许可证 | ❌ 没有 | ✅ 有 |
| 备案兼容 | ❌ 不兼容 | ✅ 兼容 |
| 国内访问速度 | 较慢 | 快（CDN加速） |
| 月费用 | 免费 | 免费（有免费额度） |
| HTTPS | ✅ | ✅ |
| 自定义域名 | ✅ | ✅ |

### CloudBase 免费额度

- 静态托管：1GB 存储 + 5GB/月 CDN 流量（够用了）
- 不需要付费，零成本跟 GitHub Pages 一样

## 备案完整流程（预计 1-2 周）

### 第一步：迁移网站到腾讯云 CloudBase（我来做）

1. 你在腾讯云控制台开通 CloudBase（静态网站托管）
2. 我帮你把网站文件全部上传上去
3. 配置自定义域名 miaotools.com.cn 指向 CloudBase
4. 测试网站在新服务器上正常运行

### 第二步：提交 ICP 备案申请（你来操作）

1. 打开 **https://console.cloud.tencent.com/beian**
2. 点击「开始备案」
3. 填写以下信息：

#### 主体信息（个人备案）
- **姓名**：你的真实姓名
- **证件类型**：身份证
- **证件号码**：身份证号
- **证件地址**：身份证上的地址
- **联系方式**：手机号 + 邮箱

#### 网站信息
- **网站名称**：妙用工具
- **域名**：miaotools.com.cn
- **网站首页 URL**：https://miaotools.com.cn
- **Hosting 提供商**：腾讯云 CloudBase（下拉选择即可）
- **网站服务内容**：选择「工具/查询类网站」
- **备注**：免费在线工具集合网站，提供二维码生成、图片压缩等实用工具

### 第三步：提交审核材料

需要准备的材料（拍照/扫描）：

1. **身份证正反面照片**（清晰，无遮挡）
2. **手持身份证照片**（部分省份要求，看具体提示）
3. **域名证书**（腾讯云后台可下载）
4. **网站真实性核验单**（腾讯云会提供模板，打印后签字拍照）

### 第四步：等待审核

- 腾讯云初审：1-3 天
- 省通信管理局审核：5-15 天
- 审核通过后会收到短信和邮件通知
- 备案号格式：粤ICP备XXXXXXXX号（取决于你的省份）

### 第五步：备案完成后

1. **在网站页脚添加备案号**（法律要求）
   - 格式：`粤ICP备XXXXXXXX号`
   - 需要链接到工信部备案查询网站
2. 重新去百度搜索资源平台验证网站 → 这次一定能通过 ✅
3. 注册百度联盟 → 接入百度广告

## 备案注意事项

1. **备案期间网站可能需要关闭**（部分地区要求，腾讯云会提示）
2. **手机号必须能接听验证电话**（管局可能会打电话核实）
3. **网站内容必须合法合规**（我们的工具站完全合规 ✅）
4. **备案信息必须真实准确**（虚假信息会被驳回）
5. **备案通过后每年需要年审**（腾讯云会提醒）
