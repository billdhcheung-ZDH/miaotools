# 妙用工具 · 部署与变现操作指南

> 一个纯静态在线工具站（10 个工具 + SEO 内容），零服务器成本，可持续扩展。
> 本指南说明如何上线、接入广告、上架数字产品，以及哪些步骤需要你本人授权。

---

## 一、项目说明

- **技术形态**：纯静态网站（HTML + CSS + 原生 JS），无后端、无数据库
- **工具数量**：10 个（二维码、图片压缩、密码生成、Base64、JSON 格式化、字数统计、颜色工具、时间戳、UUID、正则测试）
- **数据安全**：所有处理在浏览器本地完成，不上传服务器
- **托管成本**：可零成本（免费静态托管平台）
- **目录结构**：
  ```
  tools-site/
    index.html          首页（工具导航 + SEO 内容）
    css/style.css       全局样式
    js/common.js        通用脚本（导航/页脚/工具列表注入）
    tools/*.html        10 个工具页面
    robots.txt          搜索引擎抓取规则
    sitemap.xml         站点地图
  ```

## 二、上线部署（需要你授权注册账号）

网站已开发完成，部署到公网才能被访问、被搜索引擎收录、才能接入广告。
以下三种方案任选其一，**均需你注册账号**（我无法代你注册）：

### 方案 A：Vercel（推荐，最简单）
1. 注册 Vercel 账号（https://vercel.com，可用 GitHub 账号登录）—— **需你授权**
2. 将 `tools-site` 文件夹上传（拖拽到 Vercel 部署页，或关联 GitHub 仓库）
3. 几秒钟后获得一个 `xxx.vercel.app` 免费域名，网站立即上线
4. 成本：**完全免费**，流量充足

### 方案 B：GitHub Pages
1. 注册 GitHub 账号 —— **需你授权**
2. 新建仓库，上传 `tools-site` 全部文件
3. 开启 Settings → Pages，选择分支，几分钟后获得 `xxx.github.io` 域名
4. 成本：**完全免费**

### 方案 C：Netlify
1. 注册 Netlify 账号 —— **需你授权**
2. 拖拽 `tools-site` 文件夹到部署页
3. 获得 `xxx.netlify.app` 域名
4. 成本：**完全免费**

> ⚠️ 以上均涉及账号注册，请你选择一个平台后告诉我，我会一步步指导你完成部署。

## 三、绑定独立域名（可选，建议做）

免费子域名能用，但独立域名更专业、更利于 SEO 和品牌。

1. 购买域名：在阿里云/腾讯云/Namecheap 购买（.com 约 ¥70/年，.cn 需实名认证）—— **需你授权付款**
2. 在托管平台（Vercel/Netlify）添加自定义域名，按提示配置 DNS 解析
3. 平台自动签发 HTTPS 证书

> 建议域名方向：包含"工具""tool"等关键词，如 `miao tools.com`、`haoyongtool.com` 等。

## 四、接入广告变现（核心收入来源）

这是网站收入的主要来源。**需要你注册广告联盟账号并通过审核**。

### 国内：百度联盟 / 360 联盟
1. 注册百度联盟（https://union.baidu.com）—— **需你授权（需实名）**
2. 提交网站审核（需网站已上线、有一定内容、已备案或海外可访问）
3. 审核通过后获取广告代码
4. 将广告代码粘贴到页面中的 `.ad-slot` 位置（我已在每个页面预留了广告位）
5. 收入：按 CPM/CPC 结算，提现到银行卡

### 国际：Google AdSense（单价更高，推荐）
1. 注册 Google AdSense（https://adsense.google.com）—— **需你授权**
2. 提交网站审核（需有优质内容、隐私政策页面——已内置）
3. 审核通过后获取广告单元代码
4. 替换页面中的 `.ad-slot` 占位为真实广告代码
5. 收入：美元结算，满 $100 提现（电汇到银行账户）

### 接入方式
我会帮你把广告代码统一插入所有页面的广告位（`.ad-slot` 元素），你只需提供审核通过后的广告代码片段。

## 六、流量增长（决定收入的关键）

网站收入 = 流量 × 广告点击率 × 单价。流量是核心。我会通过自动化持续做以下工作：

1. **持续新增工具**：每周新增 2-3 个工具页面（更多 SEO 落地页）
2. **持续产出 SEO 文章**：为每个工具配套"使用教程""常见问题"等内容
3. **提交搜索引擎收录**：Google Search Console、百度站长平台（需你授权验证域名所有权）
4. **结构化数据**：已为每个页面配置 Schema.org 标记，利于搜索结果富文本展示

> 收录验证（Google Search Console / 百度站长）需你登录验证域名，我提供操作指引。

## 七、收入预期（诚实说明）

- **前 1-2 个月**：网站新上线，搜索引擎还在收录，流量极低，收入接近 0
- **3-6 个月**：随工具页和文章被收录，流量逐步增长，日收入 ¥10-50
- **6-12 个月**：若持续更新，日流量上千，日收入可达 ¥100-300+
- **关键**：这是复利型资产，前期投入主要为时间（我承担内容生产），后期接近被动收入

**200 元/天的目标，是一个需要 6-12 个月持续运营才能稳定达到的数字，没有合法的速成法。** 但这个资产一旦建立，可以长期产生收入，且我可以通过自动化持续扩充内容降低你的时间投入。
