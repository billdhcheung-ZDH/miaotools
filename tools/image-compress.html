图片压缩工具说明

本工具帮助你在不显著损失画质的前提下，减小图片文件体积，适用于网页上传、邮件附件、社交分享等场景。

使用方法

  1. 点击或拖拽上传图片（支持 JPG、PNG、WebP）。
  2. 拖动滑块调整压缩质量（数值越低体积越小）。
  3. 设置最大边长，超过该尺寸的图片会等比缩小（设为 0 则不缩放）。
  4. 选择输出格式后预览效果，满意后点击下载。

格式建议

  • JPEG：适合照片，体积小，不支持透明。
  • PNG：适合图标、截图，支持透明，体积较大。
  • WebP：现代格式，兼顾画质与体积，推荐用于网页。

隐私保护

图片压缩全程在你的浏览器中完成，不会上传到任何服务器，请放心使用。
