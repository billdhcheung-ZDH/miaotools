# 上线部署操作手册

> 本手册按顺序操作即可完成上线。涉及注册和付款的步骤必须你亲自完成，
> 技术配置部分可随时找我帮忙。

---

## 第一步：购买域名（腾讯云，约 ¥69/年）

### 1.1 推荐域名（含"工具"关键词，利于 SEO）

请在腾讯云逐一查询以下域名的可用性和价格，选一个能注册的：

| 域名 | 含义 | 后缀优势 |
|------|------|----------|
| `miaotools.com` | 妙工具 | .com 国际通用，SEO 最佳 |
| `haotools.com` | 好工具 | .com，简短好记 |
| `tool365.com` | 工具365 | .com，有数字感 |
| `onlinetools.cn` | 在线工具 | .cn 国内友好，需实名 |
| `toolset.cn` | 工具集 | .cn，简洁 |
| `easytools.cn` | 易工具 | .cn，好记 |

> 💡 优先选 .com（国际通用、SEO 更好）；如果 .com 都被注册了，.cn 也可以。
> 💡 不限于以上列表，你也可以在腾讯云搜索其他含 "tool" 的域名。

### 1.2 注册腾讯云账号并购买

1. 打开 **https://cloud.tencent.com** → 右上角「免费注册」
2. 用手机号注册（需验证短信）—— **需你本人操作**
3. 注册后完成**实名认证**（身份证拍照，几分钟自动通过）—— **需你本人操作**
4. 进入**域名注册**页面：https://buy.cloud.tencent.com/domain
5. 搜索你选好的域名（如 `miaotools.com`）
6. 加入购物车 → 结算（约 ¥69/年，首年可能有优惠）
7. **付款**（微信/支付宝）—— **需你本人操作**
8. .com 域名需完成**域名实名认证**（上传身份证，1-3 个工作日通过，通常更快）

### 1.3 域名实名认证通过后

告诉我你的域名是什么，我会：
- 更新网站中的 sitemap.xml、canonical 链接
- 更新 CNAME 文件
- 指导你配置 DNS 解析

## 第二步：注册 GitHub 账号（免费）

1. 打开 **https://github.com/signup**
2. 填写用户名、邮箱、密码 —— **需你本人操作**
3. 邮箱验证（收验证邮件，点链接）
4. 免费计划（Free），不需要付款

> 用户名建议：简短好记，如 `miaotools`、`yourname-tools` 等。
> 记住你的 GitHub 用户名，后面配置要用。

---

## 第三步：创建 GitHub 仓库并上传网站

### 方法 A：网页上传（最简单，推荐新手）

1. 登录 GitHub → 点右上角 **+** → **New repository**
2. 填写：
   - Repository name: `miaotools`（或你喜欢的名字）
   - 选 **Public**（必须公开才能用免费 GitHub Pages）
   - 勾选 **Add a README file**
   - 点 **Create repository**
3. 进入仓库页面 → 点 **Add file** → **Upload files**
4. 将 `tools-site` 文件夹里的**所有文件和文件夹**拖进去：
   - `index.html`、`README.md`、`robots.txt`、`sitemap.xml`、`CNAME`、`.gitignore`
   - `css/` 文件夹（含 style.css）
   - `js/` 文件夹（含 common.js）
   - `tools/` 文件夹（含 10 个 .html 文件）
5. 等待上传完成 → 底部填写 commit message → 点 **Commit changes**

> ⚠️ 注意：网页上传时文件夹结构要保持不变，`css/`、`js/`、`tools/` 是子文件夹。

### 方法 B：用 Git 命令推送（如果你有 Git 基础）

```bash
cd tools-site
git remote add origin https://github.com/你的用户名/miaotools.git
git push -u origin main
```
如果提示输入账号密码，需要在 GitHub 创建 Personal Access Token：
Settings → Developer settings → Personal access tokens → Generate new token，勾选 `repo` 权限。

## 第四步：开启 GitHub Pages

1. 进入你的仓库页面
2. 点 **Settings**（顶部标签栏）
3. 左侧菜单找 **Pages**
4. **Source** 选择 **Deploy from a branch**
5. **Branch** 选择 `main`，文件夹选 `/ (root)`
6. 点 **Save**
7. 等待 1-2 分钟，页面顶部会出现绿色提示：
   `Your site is live at https://你的用户名.github.io/miaotools/`

🎉 到这里，网站已经上线了！可以先访问那个地址确认能打开。

## 第五步：绑定自定义域名

### 5.1 在 GitHub 配置域名

1. 在仓库根目录找到 `CNAME` 文件，点编辑
2. 把内容改成你的实际域名（如 `miaotools.com`），只写域名，不加 https://
3. Commit 保存
4. 回到 Settings → Pages，在 **Custom domain** 输入你的域名 → **Save**
5. 勾选 **Enforce HTTPS**（等几分钟自动签发证书）

### 5.2 在腾讯云配置 DNS 解析

1. 登录腾讯云控制台 → **DNS 解析 DNSPod** → https://console.cloud.tencent.com/cns
2. 找到你买的域名 → 点进去
3. 添加以下解析记录：

   | 记录类型 | 主机记录 | 记录值 | TTL |
   |----------|----------|--------|-----|
   | A | @ | 185.199.108.153 | 600 |
   | A | @ | 185.199.109.153 | 600 |
   | A | @ | 185.199.110.153 | 600 |
   | A | @ | 185.199.111.153 | 600 |
   | CNAME | www | 你的用户名.github.io | 600 |

   > A 记录的 4 个 IP 是 GitHub Pages 官方服务器地址。
   > CNAME 记录把 www 子域名指向你的 GitHub Pages 地址。

4. 保存后等待 DNS 生效（通常 10 分钟 - 1 小时）
5. 浏览器访问 `https://你的域名.com` 确认能打开

### 5.3 验证 HTTPS

DNS 生效后，GitHub 会自动签发 Let's Encrypt 证书。
回到 Settings → Pages 确认 HTTPS 已启用。
访问 `https://你的域名.com` 确认锁图标出现。

## 第六步：提交搜索引擎收录

网站上线后，需要主动提交给搜索引擎收录才能获得流量。

### 6.1 百度站长平台
1. 打开 https://ziyuan.baidu.com → 登录（用百度账号）
2. 添加站点 → 输入你的域名 → 验证（CNAME 或文件验证）
3. 提交 sitemap：`https://你的域名.com/sitemap.xml`

### 6.2 Google Search Console
1. 打开 https://search.google.com/search-console → 登录 Google 账号
2. 添加资源 → 输入域名 → DNS 验证（在腾讯云加一条 TXT 记录）
3. 提交 sitemap

> 收录需要 1-4 周，期间网站内容已可正常访问。
