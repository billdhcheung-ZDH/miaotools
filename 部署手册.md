# 上线部署操作手册（已更新域名和账号）

> 域名: miaotools.com.cn
> GitHub 用户名: billdhcheung-ZDH
> 仓库地址: https://github.com/billdhcheung-ZDH/miaotools

---

## 第一步：在 GitHub 创建仓库（1 分钟）

1. 打开 https://github.com/new
2. 填写：
   - **Repository name**: `miaotools`
   - **Description**（可选）: `免费在线工具集合`
   - 选择 **Public**（必须公开才能用免费 GitHub Pages）
   - **不要勾选**任何 "Add a README file" / "Add .gitignore" / "Choose a license"
3. 点击 **Create repository**

> 创建后你会看到一个空仓库页面，不用管它，继续下一步。

---

## 第二步：运行部署脚本推送代码（2 分钟）

1. 打开文件资源管理器，进入 `tools-site` 文件夹
2. **双击运行 `deploy.bat`**
3. 脚本会自动配置 Git 并推送代码
4. 弹出浏览器时，登录你的 GitHub 账号并授权
5. 看到 "代码推送成功" 表示完成

> 如果提示推送失败，确认第一步的仓库已创建，然后重新运行 `deploy.bat`。

---

## 第三步：开启 GitHub Pages（2 分钟）

1. 打开 https://github.com/billdhcheung-ZDH/miaotools
2. 点击 **Settings**（顶部标签栏）
3. 左侧菜单找到 **Pages**
4. **Source** 选择 **Deploy from a branch**
5. **Branch** 选择 `main`，文件夹选 `/ (root)`
6. 点击 **Save**
7. 等待 1-2 分钟，页面顶部出现绿色提示：
   `Your site is live at https://billdhcheung-ZDH.github.io/miaotools/`
8. 在 **Custom domain** 输入框输入：`miaotools.com.cn`
9. 点击 **Save**
10. 勾选 **Enforce HTTPS**（等几分钟自动签发证书）

---

## 第四步：在腾讯云配置 DNS 解析（5 分钟）

1. 登录腾讯云控制台 → **DNS 解析 DNSPod**
   地址：https://console.cloud.tencent.com/cns
2. 找到 `miaotools.com.cn` 域名 → 点击进入

### 添加以下 5 条解析记录：

**A 记录 ×4（将根域名指向 GitHub Pages 服务器）**

| 记录类型 | 主机记录 | 记录类型 | 记录值 | TTL |
|----------|----------|----------|--------|-----|
| A | @ | A | 185.199.108.153 | 600 |
| A | @ | A | 185.199.109.153 | 600 |
| A | @ | A | 185.199.110.153 | 600 |
| A | @ | A | 185.199.111.153 | 600 |

**CNAME 记录 ×1（将 www 子域名指向 GitHub Pages）**

| 记录类型 | 主机记录 | 记录值 | TTL |
|----------|----------|--------|-----|
| CNAME | www | billdhcheung-ZDH.github.io | 600 |

### 操作方法：
1. 点击「添加记录」
2. 记录类型选 A，主机记录填 @，记录值填上面的 IP，TTL 默认 600
3. 重复添加 4 条 A 记录
4. 再添加 1 条 CNAME 记录：主机记录填 www，记录值填 billdhcheung-ZDH.github.io
5. 保存

> DNS 生效通常需要 10 分钟 - 1 小时。

## 第五步：验证上线

DNS 生效后：
1. 浏览器访问 `https://miaotools.com.cn` → 应该能看到工具站首页
2. 访问 `https://miaotools.com.cn/tools/qr-code.html` → 二维码生成器

## 第六步：提交搜索引擎收录

网站能正常访问后，提交收录：

### 百度站长平台
1. 打开 https://ziyuan.baidu.com → 登录
2. 用户中心 → 站点管理 → 添加网站
3. 输入 `https://miaotools.com.cn`
4. 验证域名（CNAME 验证或文件验证）
5. 验证通过后提交 sitemap：`https://miaotools.com.cn/sitemap.xml`

### Google Search Console
1. 打开 https://search.google.com/search-console
2. 添加资源 → 输入 `miaotools.com.cn`
3. DNS 验证（在腾讯云加一条 TXT 记录，按提示操作）
4. 验证通过后提交 sitemap

## ⚠️ 关于 .com.cn 域名的注意事项

`.com.cn` 域名在中国大陆有 ICP 备案要求。但 GitHub Pages 服务器在海外，
**技术上不需要备案即可访问**。但有以下风险：
- 个别运营商可能对未备案的 .cn 域名做限制（概率较低）
- 如后续遇到国内访问不稳定，可考虑：
  - 换用 .com 域名（无备案要求）
  - 或将网站迁移到国内服务器并完成备案
