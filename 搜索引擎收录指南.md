> 让百度、Google、Bing 能搜到 miaotools.com.cn
> 预计耗时：20 分钟（三个平台）

- [x] 网站已上线：https://miaotools.com.cn
- [x] sitemap.xml 已就绪：https://miaotools.com.cn/sitemap.xml
- [x] robots.txt 已就绪：https://miaotools.com.cn/robots.txt

## 一、百度搜索收录（最重要——国内流量主力）

### 第 1 步：打开百度搜索资源平台
- 地址：https://ziyuan.baidu.com
- 用百度账号登录（没有就注册一个）

### 第 2 步：添加网站
1. 点击「用户中心」→「站点管理」→「添加网站」
2. 输入：`miaotools.com.cn`
3. 选择：**HTTPS 协议**（如果 HTTPS 还没生效就先选 HTTP，生效后可改）

### 第 3 步：验证网站所有权
百度会给你 3 种验证方式，推荐用 **CNAME 验证**：

1. 选择「CNAME 验证」
2. 百度会给你一条记录，类似：
   - 类型：CNAME
   - 主机记录：`xxxxxx`（一串随机字符）
   - 记录值：`zz.baidu.com`
3. 登录腾讯云 DNS 控制台 → 找到 miaotools.com.cn → 添加这条 CNAME 记录
4. 回到百度点「完成验证」（等 1-5 分钟）

> 如果 CNAME 不方便，也可以用「文件验证」：
> 下载百度给的验证文件，放到网站根目录，然后用 GitHub 上传

### 第 4 步：提交 Sitemap
验证成功后：
1. 左侧菜单 →「数据引入」→「链接提交」
2. 找到「自动提交」→「sitemap」
3. 填入：`https://miaotools.com.cn/sitemap.xml`
4. 点击提交

### 第 5 步：手动推送（加速收录）
在「链接提交」→「手动提交」里，把以下 URL 逐条或批量填入：

```
https://miaotools.com.cn/
https://miaotools.com.cn/tools/qr-code.html
https://miaotools.com.cn/tools/image-compress.html
https://miaotools.com.cn/tools/password.html
https://miaotools.com.cn/tools/base64.html
https://miaotools.com.cn/tools/json-formatter.html
https://miaotools.com.cn/tools/word-counter.html
https://miaotools.com.cn/tools/color-picker.html
https://miaotools.com.cn/tools/timestamp.html
https://miaotools.com.cn/tools/uuid.html
https://miaotools.com.cn/tools/regex-tester.html
```

> 百度收录周期：一般 1-2 周内开始收录，完整收录可能需 1-3 个月

## 二、Google Search Console（海外流量 + SEO 数据）

### 第 1 步：打开 Google Search Console
- 地址：https://search.google.com/search-console
- 用 Google 账号登录

### 第 2 步：添加资源
1. 点击左上角下拉 →「添加资源」
2. 选择「网址前缀」
3. 输入：`https://miaotools.com.cn`
4. 点击「继续」

### 第 3 步：验证所有权
推荐用 **HTML 标记** 方式（最简单）：

1. 选择「HTML 标记」
2. Google 会给你一段 meta 标签，类似：
   ```html
   <meta name="google-site-verification" content="xxxxxxxxxxxxx" />
   ```
3. **把这段 meta 标签告诉我**，我帮你加到 index.html 的 `<head>` 里
4. 推送代码到 GitHub 后，回到 Google 点「验证」

> 也可以用 CNAME 验证（和百度类似，在 DNS 加一条 TXT 记录）

### 第 4 步：提交 Sitemap
验证成功后：
1. 左侧菜单 →「Sitemaps」
2. 输入：`sitemap.xml`
3. 点击「提交」

### 第 5 步：请求编入索引
1. 左侧菜单 →「网址检查」
2. 输入 `https://miaotools.com.cn/`
3. 点击「请求编入索引」
4. 对每个工具页面重复此操作（每天有配额限制，分几天完成）

> Google 收录周期：通常 1-3 天开始收录，1-2 周完整收录

## 三、Bing 站长工具（补充流量来源）

### 第 1 步：打开 Bing Webmaster Tools
- 地址：https://www.bing.com/webmasters
- 用 Microsoft 账号或 Google 账号登录

### 第 2 步：添加网站
1. 点击「添加站点」
2. 输入：`https://miaotools.com.cn`
3. 选择验证方式（和 Google 类似，可以用 HTML 标记或 CNAME）

### 第 3 步：提交 Sitemap
1. 验证后进入站点设置
2. 在「Sitemaps」处提交：`https://miaotools.com.cn/sitemap.xml`

## 四、后续 SEO 优化建议

### 快速见效的（我可以帮你做）：
1. **持续新增工具页面**——每个新页面 = 一个新的搜索入口
2. **每个工具页加「常见问题」FAQ**——长尾关键词，吸引精准搜索
3. **增加博客/教程板块**——如「如何用二维码做营销」「JSON 格式化常见问题」
4. **内链优化**——工具之间互相链接，提升页面权重

### 需要你做的：
1. **在社交媒体分享你的工具**——知乎、小红书、微信公众号发帖带链接
2. **友情链接交换**——找其他工具站互相链接
3. **持续检查收录情况**——在百度/Google 搜索 `site:miaotools.com.cn` 查看收录进度
